# 🍽️ Mesón do Loyo — Gestión de Turnos

## Cómo subir esta app a internet (Vercel) — GRATIS

### Paso 1 — Crear cuenta en GitHub (si no tienes)
1. Ve a https://github.com y crea una cuenta gratuita

### Paso 2 — Subir los archivos a GitHub
1. En GitHub, pulsa el botón verde **"New"** para crear un repositorio nuevo
2. Llámalo `meson-loyo-turnos`
3. Pulsa **"Create repository"**
4. Arrastra la carpeta entera `meson-loyo` a la página que aparece
5. Pulsa **"Commit changes"**

### Paso 3 — Publicar en Vercel (gratis)
1. Ve a https://vercel.com y crea cuenta con tu cuenta de GitHub
2. Pulsa **"Add New Project"**
3. Selecciona el repositorio `meson-loyo-turnos`
4. Pulsa **Deploy** — Vercel lo detecta todo automáticamente
5. En 1-2 minutos tendrás un enlace como: `meson-loyo-turnos.vercel.app`

### Paso 4 — Instalar en los móviles
**Android:**
1. Abrir Chrome → ir al enlace
2. Tocar los 3 puntitos arriba a la derecha
3. "Añadir a pantalla de inicio"

**iPhone/iPad:**
1. Abrir Safari → ir al enlace
2. Tocar el botón compartir (cuadrado con flecha)
3. "Añadir a pantalla de inicio"
